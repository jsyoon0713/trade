==================================================
  KRX 자동매매 봇 - Windows 설치 및 실행 안내
==================================================

[ 방법 1: Python 설치 후 실행 (권장) ]

준비물:
  - Python 3.11 이상 (https://www.python.org/downloads/)
    설치 시 반드시 "Add Python to PATH" 체크!
  - 이 프로젝트 폴더 전체

설치 (최초 1회):
  1. install.bat 더블클릭
  2. .env 파일에 LS증권 API 키 입력 (자동으로 열림)

매일 실행:
  1. start.bat 더블클릭
  2. 브라우저에서 http://localhost:5001 접속

--------------------------------------------------

[ 방법 2: 단독 실행파일(.exe) 빌드 ]
  (Python이 없는 PC에서 실행할 때)

1. 이 PC(Mac/Windows)에서 install.bat 실행
2. build_exe.bat 실행
3. dist\KRX자동매매\ 폴더를 대상 PC로 복사
4. 대상 PC에서:
   a. .env.example → .env 로 복사 후 API 키 입력
   b. KRX자동매매.exe 실행
   c. 브라우저에서 http://localhost:5001 접속

--------------------------------------------------

[ .env 파일 설정 항목 ]

LS_APP_KEY=발급받은_앱키
LS_APP_SECRET=발급받은_앱시크릿
LS_ACCOUNT_NO=계좌번호8자리
LS_ACCOUNT_PWD=계좌비밀번호
LS_IS_PAPER=true          ← 모의투자: true / 실거래: false

TELEGRAM_TOKEN=텔레그램봇토큰    (선택)
TELEGRAM_CHAT_ID=채팅ID         (선택)

ANTHROPIC_API_KEY=클로드API키    (뉴스/매크로 AI 분석용, 선택)
GOOGLE_API_KEY=제미나이API키     (뉴스/매크로 AI 분석용, 선택)
DART_API_KEY=DARTAPI키           (공시 분석용, 선택)

--------------------------------------------------

[ 주요 파일 구조 ]

krx-autotrader/
├── start.bat          ← 매일 실행
├── install.bat        ← 최초 1회 설치
├── build_exe.bat      ← exe 빌드 (선택)
├── .env               ← API 키 설정 (직접 작성)
├── .env.example       ← .env 작성 참고용
├── config/
│   └── settings.yaml  ← 매매 설정 (웹에서도 변경 가능)
└── logs/              ← 로그 파일
